## 自定义滚动hooks

### 安装依赖
```
yarn 
or npm install
```

## 本地启动
```
yarn start
```

## 本地编译
```
yarn build
```

## 目录结构
```
|-- src
   |-- hooks
       |-- seHorizontalScroll.ts 水平滚动hooks
       |-- useVerticalScroll.ts 垂直滚动hooks
   |-- imgs 示例图片
   |-- App.js react应用入口
   |-- App.less 主样式文件
   |-- index.js main入口文件
```
